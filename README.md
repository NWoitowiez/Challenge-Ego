# Challenge-Ego

<!-- Pasos realizados para el Challenge.

1. Instalar Angular CLI con:

npm i .g @angular/cli@latest

ng version (Para corroborar que este en la última versión)

2. Crear proyecto y ejecutarlo en una aplicación de Angular 9 con:

ng new nombreDelProyecto

cd nombreDelProyecto

ng serve (Para correr la app en el navegador)

3. Crear y organizar las carpetas de el proyecto:

Cree la carpeta components, adentro de ella cree la carpeta app, y la carpeta componentsapp.

Luego cree la carpeta interfaces que es a donde iba a definir los modelos de autos que venian en la API, en el archivo task.ts

Despues cree la carpeta services en donde importe el modulo HttpClientModule
